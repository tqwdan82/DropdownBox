Sample Custom Dropdown List

The following is a sample custom HTMLElement Dropdown Box. The custom element is basically an INPUT text field which has a DATALIST reference. The custom dropdown list has the following attributes and event handlers

    Attributes
  • id - Element Id
  • value - Value of the element. This is just an INPUT field
  • options - Dropdown list options. Sets the DATALIST in the custom element
    Event Handler Functions
  • onchange - Function that will be invoked when there is a change to the value in the Dropdown Box
